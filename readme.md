## 基于Django后端和Vue3.0前端框架的简单考勤系统

环境

+ Django 3.2.12
+ Vue 3.2.31
+ vue-cil 5.0.1
+ npm 8.3.1
+ node v16.14.0
+ ant design vue 3.1.0-rc.1 





运行

+  后端运行：

 > cd djDemo

 > python manage.py runserver

+ 前端运行

 > cd appfront

 > npm run serve 

+ 后端数据库操作脚本

> cd djDemo/attendance/managerDatadb.py
